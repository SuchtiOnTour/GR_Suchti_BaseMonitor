# Base Monitor

Base Monitor is a BepInEx plugin for **Global Rescue** by Suchti On Tour.

## Info

[Suchti Mod Manager](https://github.com/SuchtiOnTour/GR_Suchti_ModManager) is recommended for changing settings in-game.

## Features

- Compact in-game base monitoring overlay
- Base selector with single-base and ALL overview
- Department overview for vehicles and employee coverage
- Staff needs overview for hunger, toilet, shower, energy and satisfaction
- Color indicators for quick status checks
- Movable overlay window
- Configurable button position, window position, size, transparency and refresh interval
- Live config reload through config or Suchti Mod Manager
- Multilingual overlay text based on the selected Mod Manager language
- Supports English, German, French, and Polish
- No gameplay changes, display-only mod

## Installation

1. Install BepInEx for Global Rescue.
2. Download the latest `GR_Suchti_BaseMonitor` ZIP from Releases.
3. Extract the ZIP into the Global Rescue game folder.
4. The DLL should be placed here:

```text
BepInEx/plugins/GR_Suchti_BaseMonitor/GR_Suchti_BaseMonitor.dll
```

## Notes

- Tested with Global Rescue `0.3.29`.
- Config is stored next to the DLL.
- The mod can be configured in-game with `GR_Suchti_ModManager`.
- The mod meta file is used by `GR_Suchti_ModManager` for settings and overlay text localization.
